O que é HTML?

Inicialmente é importante entender que HTML não é uma linguagem de programação. O HTML nada mais é que uma forma de escrever para que os browsers entendam a estrutura do texto escrito e como ele deve ser apresentado na página . O significado da palavra HTML é uma sigla para HyperText Markup Language — Linguagem de Marcação de Hipertexto. Hipertexto, seria uma evolução do texto normal,pois foi assim que foi tratado pelo seus inventores quando surgiu a internet.

Estrutura básica de uma página HTML

A estrutura de uma página HTML é formada por: abertura da página (informa ao browser que se trata de uma página HTML - uso da tag "html"); cabeçalho (onde tem o título da página - uso da tag "head"); corpo (onde seu texto aparece no browser - uso da tag "body"); finalização (uso da tag de fechmanto "/html" - que sinaliza o final da página).

O que são tags?

As tags em HTML nada mais são que marcadores de texto que informam aos browsers como um texto está organizado.
